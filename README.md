# 小票分摊计算器
你有没有经历过这样的场景：几个人合买日用品或者拼单凑优惠，最后对着长长的小票，三个人算出三个结果？小票分摊计算器就是为了解决这个问题而生的。它基于 Streamlit 构建，打开即用，把小票上的每一个条目按照权重分摊到每个参与者头上。希望这个小工具能帮助你把算帐变得轻松一点。

## 功能介绍
在录入条目时，你可以自由填写单价、数量和折扣，并为每个商品单独指定分摊方式：所有人均摊，仅由某一位成员承担或自定义权重分配。所有已录入的条目以表格形式实时展示，并且可以直接在表格内修改任意数值。全程使用 Decimal 运算，分摊结果保留三位小数，避免四舍五入误差累积。所有条目录入完毕后，可一键导出本次账单分摊汇总与逐条明细的 Excel 表格，方便存档或发给同行人核对。

成员管理被放在了侧边栏。默认存在两名成员参与分摊，你可以随时修改名称、增减参与人数，刷新页面后自动恢复默认。

本工具支持多国货币符号显示。

## 使用方法
本项目已部署至 Streamlit Cloud，点击公开链接 https://the-receipt-splitter.streamlit.app 即可试用，无需安装。

如果想在本地运行，可以下载本仓库至本地，然后打开终端执行以下命令：

```
pip install -r requirements.txt
streamlit run app.py
```

运行后直接弹出网页，或在浏览器中输入终端显示的 Local URL 打开网页。如果你和同伴在同一个 WiFi 下，还可以让其他设备通过终端显示的 Network URL 直接访问，只要电脑保持本地运行。

## 注意事项
本工具没有接入数据库，也不保存任何历史账单，所有数据仅在浏览器会话内有效，刷新页面即重置。如需保留账单记录，请使用”导出为 Excel”功能下载到本地。
